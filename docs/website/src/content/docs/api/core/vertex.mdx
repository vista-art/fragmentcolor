## Description

A single vertex with a required position (2D or 3D) and optional properties like uv and color.



## Example

<Tabs>

<TabItem label="Rust">
<Code
code={`
use fragmentcolor::mesh::Vertex;
let v = Vertex::from([0.0, 0.0, 0.0]).with_uv([0.5, 0.5]);
`}
lang="rust"
/>

</TabItem>

<TabItem label="JavaScript">
<Code
code={`
import { Vertex } from "fragmentcolor";
const v = Vertex.from([0.0, 0.0, 0.0]).withUv([0.5, 0.5]);
`}
lang="js"
/>

</TabItem>

### Vertex::create_instance

Create an Instance from this Vertex by cloning all of its properties; the position is copied under `position2` or `position3`.

#### Example

<Tabs>

<TabItem label="Rust">
<Code
code={`
use fragmentcolor::mesh::Vertex;
let v = Vertex::from([0.0, 0.0]);
let _inst = v.create_instance();
`}
lang="rust"
/>

</TabItem>

<TabItem label="JavaScript">
<Code
code={`
import { Vertex } from "fragmentcolor";
const v = Vertex.from([0.0, 0.0]);
const _inst = v.createInstance();
`}

/>

</TabItem>

<TabItem label="Python">
<Code
code={`
