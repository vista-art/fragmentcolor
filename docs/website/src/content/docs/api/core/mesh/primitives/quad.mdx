## Description

A simple rectangle (quad) builder for 2D rendering.
Produces a Mesh with position (vec2) and uv (vec2) per-vertex.



## Example

<TabItem label="Rust">

### Quad::new()

Create a quad from two corners (min.xy, max.xy) in clip-space coordinates.

- Position layout: @location(0) vec2<f32>
- UV layout: @location(1) vec2<f32>

#### Example

<Tabs>

<TabItem label="Rust">
<Code
code={`
use fragmentcolor::mesh::primitives::Quad;
use fragmentcolor::mesh::Mesh;

_ = quad;
`}

### Quad::get_mesh()

Gets the Mesh built by this Quad.

#### Example

<Tabs>

<TabItem label="Rust">
<Code
code={`
use fragmentcolor::mesh::primitives::Quad;
use fragmentcolor::mesh::Mesh;

let quad = Quad::new([-0.5, -0.5], [0.5, 0.5]);
let mesh: Mesh = quad.get_mesh();

_ = mesh;
`}
lang="rust" meta="collapse={7-7}"
/>

</TabItem>

<TabItem label="JavaScript">
<Code
code={`
import { Quad } from "fragmentcolor";
import { Mesh } from "fragmentcolor";

const mesh = quad.getMesh();

`}

</TabItem>
