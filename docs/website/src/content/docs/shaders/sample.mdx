---
title: "Sampling"
description: Sample distributions and low-discrepancy sequences from the FragmentColor shader registry.
---

These helpers generate sample points — on the disk, on the hemisphere,
across a low-discrepancy sequence — that you can use as offsets into a
texture, as importance-sampled directions for shading, or as the input
side of a Monte Carlo estimator. Inputs are typically a `(rnd, n)` pair;
outputs are positions or directions.

Pull one in by slug and use it inside your fragment:

```rust
let main = r#"
@fragment fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let p = vogel_disk(u32(in.position.x) % 128u, 128u);
    let d = length(in.uv * 2.0 - 1.0 - p);
    return vec4<f32>(vec3<f32>(1.0 - smoothstep(0.0, 0.04, d)), 1.0);
}
"#;
let shader = Shader::new(&["sample/vogel_disk", main])?;
```

## disk_concentric

Shirley's concentric mapping from `[0, 1]^2` to the unit disk. Preserves
relative position better than naive polar sampling.

```wgsl
fn disk_concentric(rnd: vec2<f32>) -> vec2<f32>
```

## interleaved_gradient_noise

Jorge Jimenez's IGN. Pixel-coord in, scalar in `[0, 1)` out. Useful as
per-pixel decorrelation for any of the sample sequences above.

```wgsl

```

## vogel_disk

`k`-th of `n` Vogel sunflower disk samples. The golden-angle spiral with a
sqrt-r radius gives even areal coverage with no clustering.

```wgsl
fn vogel_disk(k: u32, n: u32) -> vec2<f32>
```
