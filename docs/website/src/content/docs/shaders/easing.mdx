---
title: "Easing"
description: A catalog of easing curves in the FragmentColor shader registry.
---

An **easing** function maps a normalised time `t` in `[0, 1]` to an
eased output, also typically in `[0, 1]`, that you use to animate
properties non-linearly. The standard naming convention from Robert
Penner's classic library is preserved here: `in_*` curves accelerate
from rest, `out_*` curves decelerate to rest, and `in_out_*` curves do
both — easing in for the first half of the duration and easing out
for the second.

Every entry below is a single pure WGSL function in the registry. Pull
one into a composition by slug, then call it from any stage:

```rust
let main = r#"
@fragment fn fs_main(in: VertexOutput) -> @location(0) vec4<f32> {
    let t = in.uv.x;
    let y = out_cubic(t);
    return vec4<f32>(vec3<f32>(y), 1.0);
}
"#;
let shader = Shader::new(&["easing/out_cubic", main])?;
```

Each preview below plots `y = ease(uv.x)` as a soft accent line over a
dark grid; warm tones mark `in_*`, cool tones mark `out_*`, and neutral
tones mark `in_out_*`. The vertical axis is `y = 0` at the bottom and
`y = 1` at the top; curves that overshoot or undershoot (`in_back`,
`out_back`, `out_elastic`) leave the unit box deliberately.

## in_back

Cubic curve with overshoot before the start, using the classic Robert
Penner constants.

```wgsl
fn in_back(t: f32) -> f32
```


## in_out_cubic

Cubic curve mirrored about `t = 0.5`: `4 t^3` for `t < 0.5`, the matching
reflection for `t >= 0.5`.

```wgsl
fn in_out_cubic(t: f32) -> f32
```

![easing/in_out_cubic preview](/shaders/easing/in_out_cubic.png)

## in_out_expo

Symmetric exponential ease that snaps to exact `0` and `1` at the
endpoints to avoid the residual offset of pure `2^x`.

```wgsl
fn in_out_expo(t: f32) -> f32
```

![easing/in_out_expo preview](/shaders/easing/in_out_expo.png)

## in_out_quad

Quadratic curve mirrored about `t = 0.5`: `2 t^2` for `t < 0.5`, the

```wgsl
fn in_out_quad(t: f32) -> f32
```

![easing/in_out_quad preview](/shaders/easing/in_out_quad.png)

## in_out_quart

Quartic curve mirrored about `t = 0.5`: sharper S-shape than


![easing/in_out_quart preview](/shaders/easing/in_out_quart.png)

